# 🏥 Hospital Emergency Room Dashboard

## 🔹 Short Description / Purpose  
The Hospital Emergency Room Dashboard is an interactive Power BI report designed to analyze patient data, monitor emergency room performance, and identify key factors affecting patient care and hospital efficiency. It helps healthcare professionals make data-driven decisions to improve service quality and reduce waiting times.

## 🛠️ Tech Stack  
- 📊 Power BI Desktop – Dashboard development and visualization  
- 📂 Power Query – Data cleaning and transformation  
- 🧠 DAX (Data Analysis Expressions) – Calculations and KPIs  
- 🔗 Data Modeling – Relationships between hospital data fields  
- 📁 File Format – `.pbix` and `.xlsb`  

## 📁 Data Source  
The dataset consists of hospital emergency room records, including patient details, admission status, wait time, department referrals, and treatment outcomes. The data is structured to analyze patient flow, operational efficiency, and service performance.

## 🌟 Features / Highlights  

### 📌 Business Problem  
Emergency rooms often face challenges such as overcrowding, long wait times, and inefficient resource allocation. Analyzing these issues manually is difficult and time-consuming.

Key questions addressed:
- How long are patients waiting for treatment?  
- Which departments receive the most emergency cases?  
- What are the peak hours for emergency visits?  
- How efficiently are patients being treated and discharged?  

### 🎯 Goal of the Dashboard  
To build an interactive analytics tool that:
- Monitors patient flow in the emergency room  
- Tracks waiting time and treatment efficiency  
- Identifies peak hours and workload patterns  
- Supports better hospital resource planning  

## 📊 Walkthrough of Key Visuals  

### 🔢 Key KPIs  
- Total Patients  
- Average Wait Time  
- Admission Rate  
- Discharge Rate  
- Patient Satisfaction Score  

### ⏱️ Patient Wait Time Analysis  
Shows how long patients wait before receiving treatment, highlighting delays and inefficiencies.

### 📅 Patient Visits by Time  
Displays peak hours/days when emergency cases are highest.

### 🏥 Department-wise Referrals  
Identifies which departments handle the most cases.

### 👥 Patient Distribution  
Shows breakdown based on gender, age group, or case type.

### 🚑 Admission vs Discharge Analysis  
Tracks how many patients are admitted vs discharged.

### 🎛️ Filters / Slicers  
- Department  
- Date / Time  
- Patient Category  

## 📈 Business Impact & Insights  
- Reduced wait time by identifying delays  
- Better resource allocation during peak hours  
- Improved patient care and service efficiency  
- Optimized emergency room workflow  
- Supports data-driven hospital decisions
